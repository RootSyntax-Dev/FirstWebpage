# 🌐 FirstWebpage

A basic multi-page website created using HTML.
This project demonstrates fundamental web development concepts including page navigation, media integration, and structured layout.

## ✨ Features

* 🏠 Home page with navigation
* 👤 About page
* 📞 Contact page
* 📤 Submit page
* 🖼️ Images and 🎬 video integration

## 🚀 Technologies Used

* HTML5 — Structure and content

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser
